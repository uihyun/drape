App 내부 (헤더, 카피, 워터마크, share text, fallback, 법적 페이지) + 모든 시장 App Store 디스플레이명 / 홈 화면 라벨까지 **`archelier` 단일**. Spotify·Netflix·Notion·Figma 같은 글로벌 앱 표준 패턴.

### 한국·일본 ASO 보조 키워드 (디스플레이명이 아닌 검색 키워드로만)

> 검증 (2026-05-05): KIPRIS / J-PlatPat / 일반 웹 / 앱스토어 검색 모두 동명 0건. 사용은 안전. 단 정식 디스플레이명이 아니라 ASO 키워드 차원이라 한국·일본 사용자가 한글/한자로 앱 검색 시에도 잘 잡히게 보조.

- 한국 백업: **집공방** (집 + 공방, "내 안의 작업실"). 1순위 공간담이 더 신선·간결해 우선 선택.
- 일본 백업: **アトリエ家** (atorie-ka). アトリエ가 일반명사화되어 ASO에서 묻힐 우려로 후순위. 空間師 가 더 독자적.

## archelier — 의미와 어원

**Archi**tect + at**elier** 의 합성어 (portmanteau).

- **Architect** — 그리스어 *arkhitekton*: "최고의 장인 / 설계자"
- **Atelier** — 프랑스어: "예술가·장인의 작업실, 공방"

→ **"건축가의 아틀리에"** = 설계와 손맛이 함께하는 장인의 작업실.

발음 후보:
- 영어/프랑스식: **아르셸리에** (ar-shə-LYAY) ★ 공식 발음
- 라틴/영어식: 아르켈리에 (ar-keh-LEE-er) — 비공식
- 일본 표기: **アルシェリエ**
- 한국 표기: **아르셸리에**

## 브랜드 포지셔닝

### 한 줄
> **"당신의 손끝에 있는 아키텍트 아틀리에."**
> *Your architect's atelier, in your pocket.*

### 컨셉 키워드 3개
1. **Atelier** — 장인의 작업실 (따뜻함, 손맛, 큐레이션)
2. **Architecture** — 구조, 설계, 깊이
3. **Render** — 실시간 시각화 (AI의 마법)

> "AI인데 장인 같은", "tools인데 studio 같은" 톤. 흔한 *AI Interior 앱*들과 차별화되는 포인트.

### 비주얼 가이드 (방향)

| 요소 | 방향 |
|---|---|
| **컬러** | 베이지·테라코타·차콜 + 작은 골드 액센트 (워크숍 감성) |
| **폰트** | 세리프 워드마크 (Heritage) + 산세리프 본문 (Modern) |
| **로고 모티프** | A 모노그램 + 도면 격자, 또는 atelier 창문 실루엣 |
| **사진 무드** | 자연광, 종이 질감, 블루프린트, 트레이싱지 |
| **카피 톤** | "디자이너처럼 말하지 말고, 디자이너로서 말한다" |

### 카테고리별 확장 네임스페이스

향후 라인업 확장 시 자연스럽게 붙는 형태:
- **archelier Garden** — 외부/조경
- **archelier Studio** — 실제 시공 스튜디오 매칭
- **archelier Marketplace** — 가구/마감재
- **archelier for Pros** — 전문가 모드

### archelier가 살아남은 이유

- USPTO Justia 검색 0건 — 트레이드마크 미등록
- EUIPO/KIPRIS 검색 0건
- 동명: Archelier Studio (작은 건축 스튜디오, Facebook/Twitter) 1곳만, **상표 등록 없음**
- archelier.com = $2,988 Afternic premium (살 수는 있음, 미래 옵션)
- archelier.io / archelier.app — WHOIS 무응답 → 미등록 가능성, 실시간 구매 시도로 즉시 확인 가능
- 의미가 명확 (architect + atelier) → 그 자체가 카피가 됨
- 프랑스어 어미 *-lier* (atelier, sommelier, hôtelier 계열)의 프리미엄 발음 → 가격 정당화에 유리
